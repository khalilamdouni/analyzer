# analyzer

The Analyzer project is an engine which analyses the output of [Calculator][1] reports,
This project is used as a service, it provides xml outputs to be exploited in other systems or presentation platforms.
The plaform is developed with [Clojure][2] language with the WEB API [Compojure][3]

[1]: https://github.com/khalilamdouni/calculator
[2]: http://clojure.org
[3]: http://en.wikibooks.org/wiki/Compojure

## Prerequisites

You will need [Leiningen][4] 1.7.0 or above installed.

[4]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein ring server
